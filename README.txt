IDA plugin that dumps offsets of functions and data into a C/C++ file.
Tested on: IDA Pro 9.0
---
Usage:
1. Disassemble a binary file.
2. Wait for the initial auto-analysis to finish.
3. Press `File > Script file...` or press Alt, then press F twice.
4. Select main.py.
5. Set the output file.
6. Done.
